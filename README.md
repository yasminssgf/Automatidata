# Automatidata
**Tujuan** proyek ini adalah untuk menyelidiki dan memahami data yang diberikan.

**Tujuannya** adalah menggunakan dataframe yang dibuat dalam Python, melakukan pemeriksaan singkat terhadap dataset yang diberikan, dan memberi tahu anggota tim tentang temuan Anda.

Aktivitas ini memiliki tiga bagian:

**Bagian 1:** Memahami situasi
* Bersiap untuk memahami dan mengatur dataset dan informasi taksi yang diberikan.

**Bagian 2:** Memahami data

* Buat dataframe pandas untuk pembelajaran data, analisis data eksplorasi (EDA) di masa mendatang, dan aktivitas statistik.

* Susun informasi ringkasan tentang data untuk menentukan langkah selanjutnya.

**Bagian 3:** Memahami variabel

**Informasi Ringkasan**

Setelah melihat dataset, dua variabel yang paling mungkin membantu membangun model prediksi untuk tarif perjalanan taksi adalah total_amount dan trip_distance karena variabel-variabel tersebut menunjukkan gambaran perjalanan taksi.
